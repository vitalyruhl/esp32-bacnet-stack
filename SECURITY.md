# Security Policy

## Supported Versions

This project is an early-stage ESP32/PlatformIO BACnet library. Security fixes
are handled on the current `main` branch until versioned release branches are
introduced.

## Reporting a Vulnerability

Prefer GitHub Security Advisories if available for this repository. Otherwise,
open a GitHub issue with a high-level description and avoid posting exploit
details, credentials, private network data, or exploit code publicly.

Please include the affected version or commit, the impacted component, and
enough context for maintainers to reproduce or assess the issue privately.
